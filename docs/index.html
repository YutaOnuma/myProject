MonsterEscape

  • 個人開発
  • Unity
  • C#
制作期間：2ヶ月 この作品を見る　

概要

変数も分からない状態からプログラミングを学んで、初めて制作したスマホ向け鬼ごっこゲームです。 グラフィックとリアルさを追求し、「鬼ごっこ」を出来る限り再現しました。 ただ逃げるだけではなく、鬼を倒すことが出来る「バトルモード」もあります。 ルールは逃走モードでは鬼に捕まったら、バトルモーではプレイヤーのHPが0になったらゲームオーバーです。 鬼はランダムにエリアを徘徊し、プレイヤーを見つけたら追いかけてきます。 一定距離離れると鬼は徘徊に戻ります。

制作背景

二十歳になる直前、人生を振り返って「もう一度やりたいこと」を考えた際、鬼ごっこや逃○中をしたいなぁと思いました。しかし大人が現実世界で鬼ごっこや逃○中をやるのは気が引けます。そこで、それらがゲームの中でできたら同じスリル感を味わうことができると思い、制作にあたりました。

機能

・鬼はエリアをランダムに徘徊する。個体はそれぞれ完全に徘徊ルートが異なる。逃走モードでは複数の種類の鬼が存在し、種類ごとにスピードや追跡距離などステータスが異なる。
・プレイヤーはジョイスティック操作で移動できる。バトルモードでは鬼に攻撃ができる。
・「回復」「スピードアップ」「一定時間無敵」のアイテムはランダム生成される。

苦労した点

・鬼の完全ランダム徘徊
・プレイヤーの移動と視点が独立したジョイスティック
・建物や壁越しでプレイヤーを見つけた際は「見つけていない判定」にする。

今後の実装

・鬼にダメージを与えられないバグを修正
・山や池など地形を豊にする。

制作期間：1ヶ月 リリース日：2020/12/8 この作品を見る　 GitHub

概要

インターネット詐欺を安全に擬似体験できるサービスです。不正なプログラムなどは一切ありません！ インターン先でVue.jsやAWSを利用するため、学習のアウトプットとして作成しました。また、1ヶ月という納期を決め、0から制作・リリースまで実装しました。

制作背景

近頃、安倍総理の10万円や東京オリンピックの観戦チケット、不審SMSなどに関連し、フィッシング詐欺が急増している問題があります。実際にフィッシング詐欺は年々増加しており、誰がいつ被害に遭うか分かりません。実際、私の知人も被害に遭いました。手品のタネと同様に、予め擬似体験できれば実際に被害を受けても焦らず対処できると考え、制作にあたりました。

機能

・フロントエンドはVue、バックエンドはAPIとしてRailsが担当する。
・フィッシング詐欺と偽広告が体験可能。
・体験談を投稿し、共有することが可能。
・荒れないように通報機能と管理者画面を実装。

苦労した点

・N+1問題への対処。
・オリジナルのページネーション。
・AWSでSSL化するまでのエラー対処。

今後の実装

・ログイン機能をDeviseから別の手段に変更し、完全なSPAにする。
・体験可能なネット詐欺を増やす。
